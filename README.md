## ğŸ“š What is a CNN?

A Convolutional Neural Network (CNN) is a deep learning algorithm which can take in an input image, assign importance (learnable weights and biases) to various aspects/objects in the image and be able to differentiate one from the other.

### Core Components:

1.  **Convolutional Layer**: The core building block that extracts features from the input image.
2.  **Pooling Layer**: Reduces the spatial size of the Convolved Feature to reduce the computational power required.
3.  **Fully Connected Layer**: Used to classify the image based on the features extracted by the previous layers.
